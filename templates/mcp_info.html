設定例（各AI）

ChatGPT（開発者モード）

  1. 設定 → コネクタ／開発者モード を有効化
  2. 「リモートMCPを追加」で上のエンドポイントURLを入力
  3. 認証に Authorization: Bearer {{ token or '（トークン）' }} を設定
  4. チャット内でこのコネクタを有効化して利用

Claude Code

claude mcp add --transport http accounting-support "{{ endpoint }}" \
  --header "Authorization: Bearer {{ token or '（トークン）' }}"

Gemini CLI（settings.json）

{
  "mcpServers": {
    "accounting-support": {
      "httpUrl": "{{ endpoint }}",
      "headers": { "Authorization": "Bearer {{ token or '（トークン）' }}" }
    }
  }
}
