🔌 MCP接続（各AIをこのアプリに繋ぐ）

各AI（Claude Code / ChatGPT / Gemini CLI）を、このアプリのMCPサーバーに接続します。 接続すると、AIは取り込んだ会計データやfreee/MFの全情報を読み取り、解析結果を書き戻せます。

URL内に秘密トークンを含めているので、認証ヘッダやOAuth設定は不要です。claude.ai / ChatGPT / Claude Code / Gemini すべて共通で使えます。

{{ endpoint }}
{% if not has_secret %}

設定例（各AI）

claude.ai / ChatGPT（Web版コネクタ）

  1. 設定 → コネクタ（ChatGPTは開発者モード）で「カスタム/リモートMCPを追加」
  2. URLに上の MCP接続URL をそのまま貼る
    OAuth / Client ID / 認証欄は空のままでOK。
  3. 「連携させる」→ 接続完了 → チャットで利用

Claude Code

claude mcp add --transport http accounting-support "{{ endpoint }}"

Gemini CLI（settings.json）

{
  "mcpServers": {
    "accounting-support": {
      "httpUrl": "{{ endpoint }}"
    }
  }
}
