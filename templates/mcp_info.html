🔌 MCP接続（各AIをこのアプリに繋ぐ）

各AI（Claude Code / ChatGPT / Gemini CLI）を、このアプリのMCPサーバーに接続します。 接続すると、AIは取り込んだ会計データやfreee/MFの全情報を読み取り、解析結果を書き戻せます。

Web版はURLしか渡せないため、トークン付きのこのURLを貼ってください。

{{ endpoint }}

設定例（各AI）

claude.ai / ChatGPT（Web版コネクタ）

  1. 設定 → コネクタ（ChatGPTは開発者モード）で「カスタム/リモートMCPを追加」
  2. URLに Web版コネクタ用URL（?key=… 付き）を貼る
    認証（OAuth）欄は空のままでOK。トークンはURLに含まれています。
  3. 追加して有効化 → チャットで利用

※ Web版コネクタは OAuth 方式を試みるため、ヘッダ方式（Bearer）だと 「サインインサービスに登録できません」エラーになります。上の ?key= 付きURLを使ってください。

Claude Code

claude mcp add --transport http accounting-support "{{ endpoint }}" \
  --header "Authorization: Bearer {{ token or '（トークン）' }}"

Gemini CLI（settings.json）

{
  "mcpServers": {
    "accounting-support": {
      "httpUrl": "{{ endpoint }}",
      "headers": { "Authorization": "Bearer {{ token or '（トークン）' }}" }
    }
  }
}
